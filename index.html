About

Rudriox LabTech provides focused, independent consulting around LabVantage LIMS. We help laboratories and pharma organizations with configuration, customization, development and issue resolution across core LabVantage modules and related technologies.

Key Service Areas

  • LabVantage configurations and customizations
  • Reports & labels (Jaspersoft) – design, fixes and performance tuning
  • Drivers – SDMS, LimsCI and related LabVantage integrations
  • LabVantage modules training and functional walkthroughs
  • Upgradation and installation support for LabVantage environments
  • Java / Groovy / JavaScript related issues in LabVantage customizations
  • SQL / Oracle query related issues, views and report data sources

Engagement

All work is handled on a confidential basis, with small fixes through to larger enhancements supported remotely. Commercials are defined per task or per module, depending on scope and timelines.
